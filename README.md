## Our CLI Tools

| Tool | Description | npm |
|------|-------------|-----|
| [hook](https://github.com/Fullspark-Labs/hook) | Git hooks manager | `@fullsparklabs/hook` |
| [git-undo](https://github.com/Fullspark-Labs/git-undo) | Undo git operations | `@fullsparklabs/git-undo` |

## Install All

```bash
npm install -g @fullsparklabs/hook @fullsparklabs/git-undo @fullsparklabs/port @fullsparklabs/speed-cli
```
